🔒 Política de Privacidade

Roteirize Ralph • Última atualização: Junho de 2026

A Roteirize Ralph ("nós", "nosso" ou "app") respeita a sua privacidade. Esta Política de Privacidade explica como coletamos, usamos e protegemos suas informações ao utilizar nosso aplicativo.

1. Informações que coletamos

Nosso aplicativo pode coletar os seguintes dados:

  • Dados de localização: utilizamos o GPS do seu dispositivo para definir o ponto de partida da rota e melhorar a experiência de navegação. A localização só é acessada com a sua permissão explícita.
  • Dados de entrega: informações como nome do cliente, endereço, telefone, CEP, referências e documentos (CPF/CNPJ) fornecidos por você ao importar planilhas ou inserir manualmente.
  • Endereços de e-mail (para testes): durante a fase de testes fechados, podemos solicitar seu e-mail para adicioná-lo à lista de testadores.

2. Como usamos seus dados

  • Para geocodificar endereços (converter endereços em coordenadas) usando a API do Google Maps Geocoding ou o serviço gratuito Nominatim.
  • Para otimizar a rota de entregas e exibi-la no mapa.
  • Para permitir a navegação via Waze ou Google Maps.
  • Para gerar relatórios (CSV/PDF) das entregas realizadas.

3. Armazenamento dos dados

Todos os dados de entregas, rotas e cache de geocodificação são armazenados exclusivamente no seu dispositivo, dentro do banco de dados IndexedDB do navegador WebView. Nenhum dado é enviado para nossos servidores.

4. Compartilhamento de dados

Compartilhamos dados apenas com provedores de serviços essenciais:

  • Google Geocoding API: para converter endereços em coordenadas.
  • Nominatim (OpenStreetMap): como fallback gratuito para geocodificação.
  • Waze e Google Maps: para abrir a navegação com o destino desejado.

Seus dados não são vendidos, alugados ou compartilhados com terceiros para fins de marketing.

5. Segurança

Adotamos práticas recomendadas para proteger seus dados, incluindo restrição da chave de API ao seu aplicativo Android (via SHA-1 e nome de pacote).

6. Seus direitos

Você pode a qualquer momento:

  • Limpar todos os dados do app (botão "Limpar todos os dados").
  • Revogar as permissões de localização nas configurações do dispositivo.
  • Solicitar a exclusão de qualquer dado armazenado localmente.

7. Alterações nesta política

Podemos atualizar esta política periodicamente. A versão mais recente sempre estará disponível neste link.
