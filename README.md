# Roteirize Ralph

Menos voltas. Mais entregas.

O Roteirize Ralph é um aplicativo de roteirização inteligente desenvolvido para entregadores autônomos, motoristas, transportadoras e operadores logísticos.

## Recursos

- Importação de planilhas Excel (.xlsx) e CSV
- Reconhecimento automático de colunas
- Geocodificação automática de endereços
- Otimização inteligente de rotas
- Integração com Google Maps e Waze
- Cadastro manual de entregas
- Registro de recebedor (nome e CPF/CNPJ)
- Motivo obrigatório para não entrega
- Relatórios PDF e CSV
- GPS para ponto de partida
